# project-01 — Calculator (Java Basics)

A console calculator that demonstrates core Java basics:

- variables & data types (`double`, `boolean`, `String`)
- reading input with `Scanner`
- arithmetic operators (`+ - * /`)
- methods with parameters and return values
- `switch` statement and a `while` loop
- a simple divide-by-zero guard

## Requirements
- JDK installed (`java` and `javac` on your PATH). Check with `java -version`.

## Compile & run
```bash
cd project-01
javac Calculator.java   # produces Calculator.class
java Calculator
```

## Example
```
=== Java Basics Calculator ===

Enter first number: 12
Enter an operator (+ - * /): *
Enter second number: 4
Result: 12.0 * 4.0 = 48.0
Calculate again? (y/n): n
Goodbye!
```
